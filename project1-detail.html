Vinix Career

This project resulted in a fully functional career platform that enhances user engagement and provides seamless access to job listings, mentorship programs, and skill-building resources. The design focuses on intuitive navigation and responsive interfaces to cater to diverse user needs.

Project Type: End-to-end responsive website + branding

Role: Sole UX/UI designer + brand designer (with support from my mentor, Ayelén Malaquín, and design critique facilitator, Amber KB Wilson)

Project: Industry/Academy

Tools: Figma, FigJam, Zoom, Notion

Duration: 3 months

View Hi-Fi Prototype

Project Overview

Vinix Career was developed to bridge the gap between academic learning and industry demands. The platform features personalized career paths, interactive workshops, and direct connections to employers, making it easier for users to navigate their professional journeys.

Key challenges included creating an inclusive design that accommodates diverse user backgrounds and ensuring scalability for future expansions. Through iterative user testing and feedback, the final product achieved high usability scores and positive engagement metrics.

This project not only honed my skills in UX/UI design but also taught valuable lessons in collaboration and project management within a multidisciplinary team.
