Interactive demo of Vinix Career platform



Vinix Career

Vinix Career is a career management and job portal system designed to support PT Vinix Seven Aurum’s service expansion. The project focuses on helping Vinix7 graduates explore relevant job opportunities through a clear, user-friendly experience, while laying the foundation for internal career data management. The final output was an MVP that validated both the system concept and the user experience.

Role: UX/UI Designer & Web Developer
Project Type: Industry Based-Project
Tools: Figma, HTML, CSS, JavaScript
Duration: 4 months (Aug 2025 - Dec 2025)
View Case Study View Prototype

Project Overview

The Challenge

Customers struggled to find reliable catering services that matched their event type, budget, and dietary requirements. Existing platforms lacked visual appeal for food presentation and had complex booking processes that led to high abandonment rates.

My Approach

I worked end-to-end on this project, starting from understanding user and business needs to designing and implementing the first usable version of the system. The focus was not only on visual design, but also on creating a clear flow that supports how users actually look for opportunities.

UX Process

The UX process began by mapping user flows for two roles: talent/graduates and admin. Low-fidelity wireframes were created to define page structure and interaction logic before moving into high-fidelity UI design.

Key Features

  • Event-type based menu discovery with visual galleries
  • Dietary preference filters (halal, vegan, gluten-free, nut-free)
  • Real-time availability calendar for providers
  • Integrated booking flow with deposit payment
  • Post-event feedback system to improve service quality
