Interactive demo of Catering Hub platform

Catering Hub

An all-in-one catering platform that connects customers with local catering providers for seamless event planning. Features intuitive menu browsing, dietary preference filters, and integrated booking with real-time availability.

Project Type: End-to-end responsive website + mobile app design
Role: UX/UI Designer & Front-end Developer
Industry: Food & Beverage / Catering Services
Tools: Figma, Adobe Illustrator, Webflow, GitHub
Duration: 2 months (Apr 2024 - May 2024)
View Hi-Fi Prototype

Project Overview

The Challenge

Customers struggled to find reliable catering services that matched their event type, budget, and dietary requirements. Existing platforms lacked visual appeal for food presentation and had complex booking processes that led to high abandonment rates.

My Approach

I conducted user interviews with both customers and catering providers to identify pain points in the booking journey. The solution focused on creating an emotionally engaging experience through high-quality food imagery, simplified filtering based on event type (wedding, corporate, birthday), and a transparent pricing system that builds trust.

Key Features

  • Event-type based menu discovery with visual galleries
  • Dietary preference filters (halal, vegan, gluten-free, nut-free)
  • Real-time availability calendar for providers
  • Integrated booking flow with deposit payment
  • Post-event feedback system to improve service quality

Design Process

Food-Centric Research

Analyzed food photography best practices and conducted taste-testing sessions to understand visual appeal factors that drive catering decisions.

User Journey Mapping

Mapped emotional highs and lows during event planning to create moments of delight and reduce anxiety in the booking process.

Visual Design System

Created a warm, appetizing color palette with food-friendly typography to enhance visual hierarchy and emotional connection.

Responsive Implementation

Built fully responsive interface using Webflow with mobile-first approach for on-the-go event planning.
